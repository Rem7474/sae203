Cloud Lamp Project

This project brought together many of my passions

3D modeling and printing

Objective : make 2 lamps connected together,
when the buttons are turned on one lamp, the other lamp changes to the color selected and vice versa.

3D model before printing
Software : SolidWorks

electronics and home automation

Addition of electronic components

How work electronic ? It's simple : the "heart" of the lamp is an ESP32, a microcontroller who integrates the Wifi and Bluetooth.
The ESP32 is connected to an LED strip who can control each led (WS2812B precisely).
The four potentiometers are used to select the color (red, blue, green and white values).
Finally to power the whole thing : a Li-Ion 18650 battery in 3.7V, with a capacity of 2.5Ah and a TP046 charging module (with USB-C port for charging)

Photos of the cloud lamp after 3D printing and addition of electronic components.

The code who is running in the ESP32 is written in C++ with the Arduino IDE.
The lamp is connected to the internet through wifi and it's connected to an MQTT server,
and we can control it with a web site on our smartphone.
MQTT is a protocol based on TCP/IP that allows you to send and receive messages.
When the button is pressed, the ESP32 sends a signal to the MQTT server,
which send information on the other lamp to turn on the LED strip with the selected color.
